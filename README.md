# df2
d for duplicate, f for finder, and 2 for... ahnn, whatever

# Install
```
$ pip install -r requirements.txt
$ mv df2.py ~/.local/bin/df
$ chmod +x ~/.local/bin/df
```

# Usage
```
    df2 <path>... [--quiet --output=OUTPUT --like=LIKE]
```
# Options
```
    -q --quiet
    -o --output=OUTPUT
    -h --help
    -l --like=LIKE
```
